Utilisation du plugin dcLatestVersionsLight

Les plugins "dcLatestsVersions" sont utilisés pour afficher les différentes versions téléchargeables de Dotclear.

A ce jour il existe trois types de version :

  • stable: version en cours
  • testing: développement des modifications apportées à la version (en cours)
  • unstable: prochaine version en développement

Pour activer le plugin dcLatestVersionsLight, il suffit d'aller dans vos préférences utilisateurs - tableau de bord. (colonne gauche)

Une fois activé, les différentes versions de Dotclear s'afficheront sur le tableau de bord

Pour désactiver le plugin, il suffit de retourner dans les préférences.
