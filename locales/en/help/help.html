Using dcLatestVersionsLight plugin

The"dcLatestsVersions" plugins are used to display the different downloadable versions of Dotclear.

To date there are three types of versions:

  • stable: current version
  • testing: development of changes made to version(in progress)
  • unstable: next version in development

To activate the dcLatestVersionsLight plugin, just go to your user preferences.(left column)

Once activated, the different versions of Dotclear will be displayed on dashboard

To deactivate the plugin, just go back to preferences.
